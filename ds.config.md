# Domain Sense Workspace Config

## Project Mode

DS mode: standard.

## Executable Implementation Boundary

**[ds:project.source-files-boundary]**\
Весь исполнимый код проекта находится в каталоге `web-canvas/`. Файлы вне `web-canvas/` не считаются исполнимой частью игры и не должны анализироваться как реализация игрового поведения, если отдельное sense-требование явно не расширяет эту границу.

## Local Verification

**[ds:project.local-test-command]**\
Команда локальной проверки для `web-canvas/` пока не зафиксирована. До фиксации агент не должен выбирать другой способ запуска или выводить требования к реализации из временной команды проверки.

## Generated / Temporary Artifacts

**[ds:project.publish-artifacts]**\
Механика подготовки публикуемого кода пока не предусмотрена. Временные файлы, сборочные артефакты и упаковки, возникающие при локальной проверке или работе инструментов, не являются исполнимой реализацией проекта и не должны отслеживаться как часть sense-модели, если отдельное sense-требование не вводит их явно.
